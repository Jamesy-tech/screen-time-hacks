# Keylogger Setup

## Installation
`git clone https://github.com/GiacomoLaw/Keylogger && cd keylogger/mac`  
`make && make install`

## Create File
`touch ~/logfile.txt`

## Start Logging
`keylogger ~/logfile.txt`

## File Location
`/Users/yourname/logfile.txt`

## Start on Startup
`sudo make startup`

## Uninstall
`sudo make uninstall`
